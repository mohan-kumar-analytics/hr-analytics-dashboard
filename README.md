## Project Overview
This project was developed to analyze HR data using SQL and Power BI. The objective was to create interactive dashboards, evaluate workforce metrics, and generate actionable insights to support HR decision-making.

## Dataset
- Total Records: 10000
- Total Columns: 13
- Source: HR Employee Database

## Tools & Technologies
- SQL (Data Extraction & Analysis)
- Power BI (Dashboard Development)
- Microsoft Excel (Data Validation)

## Dashboard Features
- Employee Count KPI
- Remote vs HQ Employee Percentage
- Average Employee Age
- Hiring Trend by Year
- Gender Distribution Analysis
- Employees by Department
- Employees by State
- Employees by Race

## Key Insights
- Analyzed workforce distribution across departments and locations.
- Evaluated hiring trends to identify recruitment patterns.
- Compared remote and HQ workforce distribution.
- Assessed employee diversity using gender and race analysis.
- Identified workforce demographics using age distribution.

## Business Recommendations
- Improve workforce planning based on hiring trends.
- Optimize remote and onsite workforce allocation.
- Support diversity and inclusion initiatives.
- Enhance strategic HR decision-making using dashboard insights.

## Project Outcome
Developed an interactive HR Analytics dashboard that transformed raw employee data into meaningful business insights using SQL and Power BI.
